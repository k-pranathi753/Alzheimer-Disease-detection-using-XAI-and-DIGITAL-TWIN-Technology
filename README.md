# Alzheimer-Disease-detection-using-XAI-and-DIGITAL-TWIN-Technology
Deep learning–based Alzheimer’s Disease detection system using MRI brain scans and ResNet50. Integrates Explainable AI techniques like Grad-CAM, LIME, and SHAP for transparent predictions, along with a Digital Twin model to visualize disease risk progression over time.
This repository contains a deep learning–based framework for detecting and analyzing Alzheimer’s Disease from MRI brain scan images using the OASIS dataset. The project leverages a pretrained ResNet50 model for image classification and integrates Explainable AI (XAI) techniques such as Grad-CAM, LIME, and SHAP to improve model interpretability and clinical transparency.

🚀 Features:
🔍 MRI Image Classification using ResNet50
🧠 Alzheimer’s Disease Detection with deep learning
📊 Explainable AI Integration
Grad-CAM for visual attention maps
LIME for local image explanations
SHAP for feature importance analysis

📂 OASIS MRI Dataset Support
📈 Digital Twin Risk Progression Simulation
⚡ Built with PyTorch and TorchVision

📁 Project Workflow
Load and preprocess MRI brain scan images
Train a pretrained ResNet50 model
Evaluate classification performance
Generate explainability visualizations using:
Grad-CAM
LIME
SHAP
Simulate patient disease progression using a Digital Twin model

📊 Explainable AI Components
Grad-CAM: Highlights important image regions influencing the model’s prediction.
LIME: Provides interpretable local explanations for MRI image classifications.
SHAP: Analyzes feature importance for clinical and tabular data predictions.

🧬 Digital Twin Simulation
The project includes a Digital Twin module to simulate Alzheimer’s disease risk progression over time using prediction scores.
